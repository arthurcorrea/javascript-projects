Perguntas Frequentes

Como funciona?

Originalmente, há uma class 'question-text' que está com 'display: none' em cada <div> de resposta.

Em cada <article> (pergunta) há um <button> (+) que possui a classe 'question-btn', e é adicionado um addEventListener em cada botão que monitora os clicks, fazendo uma verificação para analisar se o elemento clicado já possui uma classe chamada 'show-text', a qual faz com que o texto apareça.

Se não houver, ele adiciona a classe 'show-text' ao <article>, fazendo com que esse texto de resposta apareça. Se houver, ele remove a classe 'show-text', fazendo com que o texto se esconda.

Por que esse projeto é relevante?

Este projeto pode ser um estudo de base para começar a implementar SPA's (Single Page Applications) em projetos, além de ser um reforçador do método addEventListener, que é um importante verificador de eventos/inputs feitos pelo usuário.

O que aprendi com esse projeto?

  • Propriedade classList, que permite acessar os métodos de manipulação de classes CSS com JavaScript.
  • Método addEventListener e como funciona sua callback.
  • Método toggle e sua interação com classes CSS (alterna entre remover e adicionar dependendo da condição do elemento).
  • Pegar todos os elementos com determinado seletor e percorrê-los com um forEach.

Sobre o projeto

Seu funcionamento, importância e aprendizado

Funcionamento

Para cada botão é passado um "data-id" e para cada div é passado o id correspondente, fazendo uma ligação entre o botão e a div. Depois disso, é verificado pelo JavaScript, por meio de um addEventListener, o target do evento, e com isso é possível capturar o id que foi clicado. Após ser verificado que o id corresponde à aba, a classe "active" é ativada ou desativada, possibilitando a alternância das abas e dos textos.

Importância

É um projeto importante, pois é mais uma prática com monitoramento de eventos, além de usar bastante manipulação de classes CSS.
